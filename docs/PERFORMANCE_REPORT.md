## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 시스템명 | 정보통신사업부 스마트 견적서 자동화 |
| 접속 방법 | URL 접속 (PC·모바일 모두 가능) |
| 핵심 기능 | ① 견적 자동 계산 &nbsp;② PDF 즉시 생성 &nbsp;③ DB 자동 저장 |
| 연동 시스템 | 카카오 주소 API · 건축물대장 API · Airtable DB |
| 배포 환경 | Railway.app (클라우드, 24시간 접속 가능) |

### 도입 전 vs 도입 후

| | 도입 전 | 도입 후 |
|--|---------|---------|
| 건물 정보 | 인터넷 검색 후 수기 입력 | 주소 선택 → 자동 입력 |
| 금액 계산 | 엑셀 수식 직접 작성 | 연면적 입력 → 즉시 자동 계산 |
| PDF 생성 | 엑셀 → PDF 변환 → 파일명 정리 | 버튼 1번 → 10초 |
| DB 저장 | 별도 시트에 수기 기록 | 자동 저장 (0초) |
| 작업 가능 장소 | 사무실 PC (엑셀 설치 필요) | URL만 있으면 어디서든 (모바일 포함) |
| 견적 이력 관리 | 개인 파일로 산재, 통합 조회 불가 | Airtable DB 자동 누적 |

## 2. 업무 효율화 성과

### 2-1. 견적서 1건당 시간 절감

| 작업 항목           | 도입 전         | 도입 후       | 절감률          |
| --------------- | ------------ | ---------- | ------------ |
| 건물 정보 조회·입력     | 5~10분        | 30초        | **약 95%**    |
| 금액 계산           | 10~20분       | 즉시         | **100%**     |
| PDF 저장 및 파일명 정리 | 5분           | 10초        | **97%**      |
| DB 기록           | 5분           | 0초 (자동)    | **100%**     |
| **합계**          | **약 25~40분** | **약 2~3분** | **약 90~92%** |

---

### 2-2. 누적 절감 시간 (연간 환산)

> 견적서 생성 건수에 따른 시나리오별 절감 효과

| 일 처리 건수 | 일 절감 시간 | 월 절감 시간 | 연 절감 시간 | 시급 25,000원 기준 연간 절감 인건비 |
|------------|-----------|-----------|-----------|----------------------------------|
| 하루 3건 | 약 1.1시간 | 약 22시간 | 약 264시간 | **약 660만 원** |
| 하루 5건 | 약 1.8시간 | 약 37시간 | 약 444시간 | **약 1,110만 원** |
| 하루 10건 | 약 3.7시간 | 약 74시간 | 약 888시간 | **약 2,220만 원** |

> 절감된 시간은 단순 삭감이 아닌 **영업·고객 관리 등 더 가치 있는 업무에 재투입** 가능한 자원입니다.

---

### 2-3. 휴먼에러 원천 차단

| 오류 유형 | 도입 전 | 도입 후 |
|----------|---------|---------|
| 연면적·주소 오기입 | 수기 입력으로 빈번 발생 | 건축물대장 API 자동 입력 → **0건** |
| 금액 계산 오류 | 엑셀 수식 실수 가능 | 코드로 고정된 계산 로직 → **0건** |
| 파일명 규칙 불일치 | 담당자마다 상이 | 시스템이 자동 생성 → **100% 통일** |
| DB 누락 기록 | 바쁠 때 미기록 빈번 | PDF 생성과 동시 자동 저장 → **누락 없음** |

> **실수로 인한 재작업 시간과 고객 신뢰 손상 리스크를 구조적으로 제거**

---

### 2-4. 데이터 자산화

- **기존**: 견적 이력이 개인 PC의 엑셀 파일에 산재 → 퇴사 시 데이터 소멸 위험
- **도입 후**: 고객·건물·금액·서비스 항목·담당자 정보가 Airtable DB에 자동 누적
- **활용 가능성**:
  - 기존 고객 재방문 시 이전 견적 즉시 조회·비교
  - 연간 견적 건수, 금액 규모, 고객사 현황 분석
  - 영업 파이프라인 데이터로 활용 가능
  - 인수인계 시 데이터 손실 없이 즉시 인계

---

### 2-5. 장소·기기 독립성 확보

- **기존**: 사무실 PC + 엑셀 설치 환경에서만 작업 가능
- **도입 후**: 인터넷이 되는 곳이라면 어디서든 접속 가능
  - 외부 점검 현장에서 모바일로 즉시 견적서 생성
  - 출장 중 고객 요청에 즉각 대응
  - 사무실 부재 중에도 팀원 누구나 동일한 품질로 작업 가능

---

### 2-6. 표준화 및 온보딩 효율

- 견적서 양식이 시스템에 고정 → **담당자에 따른 품질 편차 없음**
- 신입·비숙련 직원도 시스템 안내에 따라 즉시 견적서 작성 가능
- 기존 교육 소요 시간(엑셀 수식·양식 설명 등) 대폭 단축

### 3-5. 내부 자동화 담당자의 역할 확장

- **기존 역할**: 데이터 정리, 보고서 작성, 단순 운영 업무
- **이번 성과**: DB 설계 → UI 기획 → 개발 → 클라우드 배포까지 **단독 완수**
- AI를 도구로 활용해 전공 지식 없이 **실무 수준의 시스템 직접 구축**
- 조직 내 AI 활용 선례 창출 → **타 부서·유사 업무에 적용 가능한 모델 제시**

## 4. 정성적 성과

- **고객 대응 속도 향상**: 영업 담당자가 현장에서 즉시 견적서 제출 가능
- **협상 유연성 증가**: 조건 변경 시 실시간 수정·즉시 재출력 → 협상 자리에서 바로 대응
- **브랜드 일관성**: 표준화된 양식 강제 적용으로 모든 견적서 품질 동일
- **리스크 분산**: 특정 담당자 의존도 제거 → 누구든 동일한 결과물 산출
- **조직 디지털화**: 구두·종이 기반 프로세스를 데이터 기반으로 전환
