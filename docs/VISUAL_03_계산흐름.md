# 견적 계산 로직 흐름
> 금액은 어떻게 자동으로 계산되는가

```mermaid
flowchart TD
    A(["📐 연면적 입력"])
    B["① 등급 자동 판별<br/>초급 · 중급 · 고급 · 특급"]
    C["② 조건표 조회<br/>성능 · 유지 · 위탁 기준 금액"]
    D["③ 조정 적용<br/>배수 × 1 − 할인율<br/>항목 ON / OFF"]
    E["④ 항목별 최종 금액"]
    F{{"부가세 포함?"}}
    G(["🎯 연간 합계 · 월 납부액"])

    A --> B --> C --> D --> E --> F
    F -->|"포함"| G
    F -->|"별도"| G

    style B fill:#dbeafe,stroke:#3b82f6
    style C fill:#dcfce7,stroke:#22c55e
    style D fill:#fef9c3,stroke:#eab308
    style E fill:#fce7f3,stroke:#ec4899
    style G fill:#fce7f3,stroke:#ec4899
```

---

## ① 등급 판별 기준

| 연면적 | 등급 |
|--------|------|
| 5,000 ~ 15,000 ㎡ | 초급 |
| 15,000 ~ 30,000 ㎡ | 중급 |
| 30,000 ~ 60,000 ㎡ | 고급 |
| 60,000 ㎡ 이상 | 특급 |
