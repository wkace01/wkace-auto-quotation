### 수식 로직

#### 🔑 핵심 입력 트리거: `E9` 셀 (고객명/대상처명)
사용자가 이 셀에 값을 입력하면, 아래 모든 VLOOKUP이 자동으로 실행됩니다.

#### 견적일 자동 입력
```
E8 = =TODAY()
```
> 파일을 열거나 출력하는 **당일 날짜**가 자동으로 표시됩니다.

#### 대상처 정보 자동 로드 (`대상처 목록` 시트 연동)
`E9`(대상처명)를 Key로 `대상처 목록` 시트를 VLOOKUP하여 건물 정보를 자동 완성합니다.

| 셀 | 항목 | 수식 | 참조 열(대상처 목록 기준) |
|----|------|------|---------------------------|
| `J14` | 대상물 (건물명) | `=E9` | - (직접 참조) |
| `J15` | 주소 | `=IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z756,2,FALSE),"")` | 2번째 열 |
| `J16` | 사용승인일 | `=IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z756,6,FALSE),"")` | 6번째 열 |
| `J17` | 주용도 | `=IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z756,3,FALSE),"")` | 3번째 열 |
| `W17` | **연면적 (㎡)** ⭐ | `=IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z756,4,FALSE),"")` | 4번째 열 |
| `J18` | 담당자명 | `=IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z756,11,FALSE),"")` | 11번째 열 |
| `W18` | 담당자 연락처 | `=IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z756,12,FALSE),"")` | 12번째 열 |

> ⭐ `W17` (연면적)은 2.1, 2.2, 2.3, 4번 시트 모든 금액 산출의 **기준값**이 됩니다.

#### 담당자 연락처 이중 조회 (`담당자 연락처` 시트 연동)
```
Q12(담당자명) = =IFERROR(VLOOKUP(E9,'대상처 목록'!C1:Z446,14,FALSE),"")
X12(전화번호) = =IFERROR(VLOOKUP(Q12,'담당자 연락처'!A1:B29,2,FALSE),"0")
```
> `대상처 목록`에서 담당자명을 먼저 찾고, 그 이름을 다시 `담당자 연락처` 시트에서 검색하여 전화번호를 가져오는 **이중 VLOOKUP** 구조입니다.

#### 금액 집계 (세부 산출 시트 연동)
세부 산출 시트들의 최종 합계 금액을 가져와 집계합니다.

| 셀 | 항목 | 수식 |
|----|------|------|
| `T23` | 성능점검 금액 | `='2.1 성능점검 산출내역'!H17` |
| `T24` | 유지점검 금액 | `='2.2 유지점검 산출내역'!H17` |
| `T25` | 선임 위탁 금액 | `='2.3 선임 산출내역'!H17` |
| `T26` | 합계 | `=SUM(T23:X25)` |
| `T27` | 최종 견적금액 [년] | `=T26` |
| `T28` | 월 견적금액 | `=T27/12` |

#### 등급 표시

```
Y25 = =IFERROR(INDEX('견적 조건표'!C4:C9, MATCH('1. 견적서'!W17,'견적 조건표'!B4:B9, 1)), "Invalid Floor Area")
```
> 연면적(`W17`)에 해당하는 건물 **등급**(초급/고급/특급)을 `견적 조건표`에서 찾아 표시합니다.

## 📊 시트 2: `2.1 성능점검 산출내역`

### 역할
- 성능점검 업무 수수료를 **직접인건비 → 직접경비 → 제경비 → 기술료 → 소계 → 조정** 순서로 산출합니다.
- 이 시트의 최종값 `H17`이 `1. 견적서`의 `T23`으로 연결됩니다.

---

### 계산 구조 (순서)

```
1) 직접인건비: 등급별 수량 × 노임단가
2) 직접경비:  직접인건비 × 10%
3) 제경비:   직접인건비 × 110%
4) 기술료:   (직접인건비 + 제경비) × 20%
5) 소계:     1) + 2) + 3) + 4)
6) 조정금액:  합계금액 - 소계 (역산)
7) 합계금액:  견적 조건표에서 연면적 기준으로 가져온 목표금액
```

---

### 직접인건비 산출 (`H6~H9`)

#### Step 1: 투입 인력 수량 결정 (INDEX & MATCH → VLOOKUP)

먼저 `1. 견적서!W17`(연면적)을 기준으로 `견적 조건표`에서 **성능점검 인력**을 조회합니다.

| 셀 | 내용 | 수식 |
|----|------|------|
| `N6` | 성능점검 등급 기준 | `=INDEX('견적 조건표'!C4:C9, MATCH('1. 견적서'!W17,'견적 조건표'!B4:B9, 1))` |
| `O6` | 성능점검 인력 수량 | `=INDEX('견적 조건표'!K4:K9, MATCH('1. 견적서'!W17,'견적 조건표'!B4:B9, 1))` |

이후 L열의 등급 라벨(특급/고급/중급/초급)을 Key로 N5:O6 표를 VLOOKUP하여 해당 등급 수량을 E열에 배치합니다.

| 셀 | 등급 | 수식 | 노임단가(G열) |
|----|------|------|---------------|
| `E6` | 특급 | `=IFERROR(VLOOKUP(L4,N5:$O$6,2,FALSE),"0")` | `330,713` |
| `E7` | 고급 | `=IFERROR(VLOOKUP(L5,N5:$O$6,2,FALSE),"0")` | `301,470` |
| `E8` | 중급 | `=IFERROR(VLOOKUP(L6,N5:$O$6,2,FALSE),"0")` | `272,298` |
| `E9` | 초급 | `=IFERROR(VLOOKUP(L7,N5:$O$6,2,FALSE),"0")` | `234,973` |

#### Step 2: 등급별 인건비 계산
```
H6 = E6 × G6  (특급 수량 × 노임단가)
H7 = E7 × G7  (고급 수량 × 노임단가)
H8 = E8 × G8  (중급 수량 × 노임단가)
H9 = E9 × G9  (초급 수량 × 노임단가)
H10 = SUM(H6:H9)  ← 직접인건비 합계
```

---

### 간접비용 산출 (`H11~H14`)
| 셀 | 항목 | 산식 | 비율 |
|----|------|------|------|
| `H11` | 직접경비 | `=H10 × 0.1` | 직접인건비의 10% |
| `H12` | 제경비 | `=H10 × 1.1` | 직접인건비의 110% |
| `H13` | 기술료 | `=(H10 + H12) × 0.2` | 직접인건비+제경비의 20% |
| `H14` | 소계 (5) | `=SUM(H10:H13)` | 1+2+3+4 합계 |

---

### 최종 금액 및 조정 (`H15, H17`)
```
H17 = =INDEX('견적 조건표'!I4:I9, MATCH('1. 견적서'!W17,'견적 조건표'!B4:B9, 1))
    → 연면적에 해당하는 '성능점검 목표 금액'을 견적 조건표에서 직접 참조

H15 = =H17 - H14
    → 조정금액 = 목표금액 - 원가산출 소계 (역산으로 적정이윤 확인)
```

## 📊 시트 3: `2.2 유지점검 산출내역`

### 역할
- **구조는 `2.1 성능점검`과 완전히 동일**합니다. 다른 점은 `견적 조건표`에서 참조하는 열이 성능점검 데이터가 아닌 **유지점검 데이터**라는 것입니다.

---

### 성능점검 시트와의 차이점

| 항목 | 2.1 성능점검 | 2.2 유지점검 |
|------|-------------|-------------|
| 인력 수량 참조 | `견적 조건표!K열` (성능점검 인력) | `견적 조건표!L열` (유지점검 인력) |
| 최종 금액 참조 | `견적 조건표!I열` (성능점검 금액) | `견적 조건표!H열` (유지점검 금액) |
| VLOOKUP 테이블 | `N5:$O$6` | `Q5:$R$6` |

```
H17 = =INDEX('견적 조건표'!H4:H9, MATCH('1. 견적서'!W17,'견적 조건표'!B4:B9, 1))
    → 연면적에 해당하는 '유지점검 연간 금액'
```

> 나머지 직접인건비, 직접경비, 제경비, 기술료, 소계, 조정금액 계산 구조는 2.1과 동일합니다.

## 📊 시트 4: `2.3 선임 산출내역`

### 역할
- 유지관리자 **비상주 위탁 선임** 수수료를 계산합니다.
- 성능·유지점검과 다르게, "해당 건물이 몇 등급 선임을 1명 필요로 하는가?"를 판단하는 로직이 추가됩니다.

---

### 핵심 로직: 선임 자격 판단 (`E6~E9`)

연면적에 따른 **법정 선임 자격** 조견표:

| 연면적 | 선임 자격 |
|--------|-----------|
| 6만㎡ 이상 | 특급 1명 |
| 3만~6만 미만 | 고급 1명 |
| 1만5천~3만 미만 | 중급 1명 |
| 1만~1만5천 미만 | 초급 1명 |
| 5천~1만 미만 | 초급 1명 |

`T5:U6` 테이블에서 연면적별로 해당 등급에 `1`, 나머지에 `0`을 설정하고:
```
E6 = =IFERROR(VLOOKUP(L4, T5:$U$6, 2, FALSE), "0")  ← 특급 필요 여부 (1 or 0)
E7 = =IFERROR(VLOOKUP(L5, T5:$U$6, 2, FALSE), "0")  ← 고급 필요 여부
E8 = =IFERROR(VLOOKUP(L6, T$5:$U6, 2, FALSE), "0")  ← 중급 필요 여부
E9 = =IFERROR(VLOOKUP(L7, T$5:$U6, 2, FALSE), "0")  ← 초급 필요 여부
```

---

### 기간 및 단가 결정 (`F열, G열`)
```
F6 = =IF(E6=1, 12, "0")
    → 해당 등급 선임이 필요하면(E=1) 12개월, 아니면 0

G6 = =IF(E6=1, INDEX('견적 조건표'!$F$4:$F$9, MATCH('1. 견적서'!$W$17,...)), 0)
    → 해당 등급 선임이 필요하면 견적 조건표에서 월 선임 단가를 가져옴, 아니면 0
```

---

### 선임 수수료 계산 (`H열`)
```
H6 = E6 × F6 × G6  (투입인원 × 기간[월] × 노임단가)
H7 = E7 × F7 × G7
H8 = E8 × F8 × G8
H9 = E9 × F9 × G9
H10 = SUM(H6:H9)  ← 직접인건비 합계
```

---

### 간접비용 및 최종 합계
선임 산출내역의 경우, **직접경비/제경비/기술료 비율이 모두 0**으로 설정되어 있습니다. 즉, 직접인건비가 곧 최종 금액이 됩니다.
```
G11 = 0  (직접경비율)
G12 = 0  (제경비율)
G13 = 0  (기술료율)

H14 = SUM(H10:H13) = H10  (나머지가 0이므로)
H17 = =H14  (소계 = 최종 합계금액)
H15 = =H17 - H14 = 0  (조정금액 없음)
```

## 📋 시트 5: `3. 업무처리 절차`

### 역할
- 정보통신설비 유지보수관리 업무의 **법적 처리 흐름도**를 보여주는 안내 시트입니다.
- **수식 없이 고정 텍스트만** 존재하며, 법적 근거와 절차를 시각적으로 설명합니다.

---

### 주요 내용 구조 (텍스트 고정)
| 구분 | 내용 |
|------|------|
| 좌측 흐름 | **정보통신설비 유지보수관리** 절차 |
| 우측 흐름 | **정보통신설비 성능점검** 절차 |
| 공통 시작 | 관리주체 [정보통신설비 소유자 또는 관리자] |

**유지보수관리 단계:**
1. 유지보수관리 자료 구비 (기준 제6조)
2. 자체관리 or 관리위탁(공사업자)
3. 지자체에 선·해임 신고
4. 유지보수관리 및 성능점검 계획 수립 (최초 점검 실시 전)
5. **정보통신설비 유지보수관리 실시 - 반기별 1회 이상 기록** (기준 제8조)

**성능점검 단계:**
1. 성능점검 자료 구비 (기준 제6조)
2. 자체점검 or 점검대행(공사업자·용역업자)
3. 성능점검 계획 수립 (최초 점검 실시 전)
4. **정보통신설비 성능점검 실시 - 매년 1회 이상** (기준 제10조 제1항)
5. **성능점검표 기록 및 보존** (기준 제10조 제4항)
6. 시·군·구청장 요청 시 제출 (기준 제10조 제5항)

## 📊 시트 6: `4. 성능점검 수량내역`

### 역할
- 건물에 설치된 각 정보통신 설비 항목별 **점검 수량**과 **기준인원**을 입력하면, **조정계수**를 반영한 투입인원을 최종 산출합니다.
- 이 시트의 총 투입인원이 `2.1 성능점검 산출내역`의 인력 배분 기준이 됩니다.

---

### 조정계수 로드 (`F4`)
```
F4 = =IFERROR(INDEX(조정계수!B4:B15, MATCH('1. 견적서'!W17, 조정계수!C4:C15, 1)), "Invalid Floor Area")
```
> `1. 견적서!W17` (연면적)을 `조정계수` 시트에서 검색하여 해당 연면적 구간의 **조정계수**를 가져옵니다.  
> 이후 모든 행(F5~F38)은 `=$F$4`로 동일한 값을 참조합니다.

---

### 투입인원 합계 계산 (`G열`)
수량(C) × 기준인원(E) × 조정계수(F)를 각 행마다 계산합니다.
```
G5 = C5 × E5 × F5  (케이블설비 수량 × 0.29 × 조정계수)
G6 = C6 × E6 × F6  (배관설비 수량 × 0.58 × 조정계수)
...
G38 = C38 × E38 × F38 (통신 접지설비)

G39 = =SUM(G5:G38)         ← 전체 합계 (소수점 포함)
G40 = =ROUNDDOWN(G39, 0)   ← 투입인원 (소수점 버림, 정수)
```

> 점검 대상이 아닌 설비는 **C열(점검수량)을 `0`으로** 설정하면 자동으로 합계에서 제외됩니다.
